更新日誌

PromptBox 的每一次演進都記在這裡。想第一時間拿到新版本，請關注 GitHub Releases。

v3.2.3

2026-07-11 最新版本

UI 視覺與操作優化包——讓重要的更顯眼、高頻的更省力、高風險的有防呆。

  • 👁️密碼可視切換：所有密碼欄位內建顯示 / 隱藏切換，輸入前先確認，降低「打錯密碼 → 永久鎖死」的風險
  • 🖱️操作按鈕加大：卡片的鎖定 / 編輯 / 複製按鈕放大至 44px 觸控目標，更好點
  • 🏷️標籤灰階降噪：移除七彩標籤配色，標題與內容回到視覺主角
  • 🧹標籤一鍵重設：側欄新增「重設」按鈕，多選標籤一鍵清空
  • 🟠「橘紅 = 機密」視覺語意：機密相關元素全站統一改用橘紅色，一眼識別
  • 🃏預設檢視改為網格模式；編輯視窗顯示卡片 #id；串聯資料夾移入固定導覽

v3.2.2

2026-07-10

機密卡片變數支援——解鎖後恢復完整變數工作流，安全不打折。

  • 🔣解鎖後變數面板：機密卡片解鎖後可正常解析 {{變數}}、填值、複製代入
  • 💾變數預設值可更新：機密卡解鎖後可儲存變數預設值（附明確警示）
  • 💬設密提示：含變數的卡片設為機密時，提前告知變數面板需解鎖後才可使用
  • 🛡️修復資料保護問題：僅更新變數時不再誤清其他變數值

v3.2.0 – v3.2.1

2026-06-23

串聯槽與變數值繼承，加上一輪看不見但很重要的內部品質強化。

  • 🧩可編輯串聯槽：串聯卡片改用專屬輸入槽，邊打 {{@prompt:N}} 邊即時展開預覽
  • 🧬變數值繼承：被引用卡片已填的變數值自動帶入組合卡當預設；同名不同值會留空並提示
  • 🎛️強型別設定 GUI：變數可指定型別（數字、布林、列舉），MCP 呼叫時參數受驗證
  • ⚙️前端架構重構與效能優化，操作更順暢（對外行為不變）

v3.1.0

2026-06-10

卡片串聯與機密釋放——Prompt 可以組合了，機密可以在你授權下安全餵給 AI。

  • 🔗卡片串聯：在串聯資料夾用 {{@prompt:N}} 遞迴組合多張卡片，含循環偵測與深度上限；鎖定 / 機密卡片無法被間接引用
  • 🔓機密釋放（Consent）：機密卡片可設釋放策略（never / once / session），AI 請求時彈出授權窗，你輸密碼授權後「解一次、餵一次、不快取」
  • 🧾釋放稽核：每次機密釋放（允許 / 拒絕 / 逾時）都留下紀錄，MCP 面板可檢視

v3.0.0

2026-06-03 重大版本

升級為「機密 Context 保險庫」——整庫加密、AI 精準取用、機密永不外洩。

  • 🔐靜態加密：整個資料庫以 SQLCipher（AES-256）加密，金鑰由 OS Keychain 保護
  • 🛠️MCP Tools 化：每張未鎖定卡片自動成為 MCP Tool，AI 帶參數呼叫、取得編譯後的純淨字串
  • 🔒資料鎖：卡片一鍵鎖定，對 AI 完全隱藏
  • 🔐機密卡片：自訂密碼的卡片級加密，GUI 也需密碼查看，AI 無法取得
  • 🪙具名多 Token：MCP Token 可具名、可個別撤銷、含速率限制
  • 🔄更新方式調整：改為「發現新版 → 提示 → 手動下載」，不再無聲自動安裝
warning

升級注意：從 v2.x 升級時，舊明文資料庫會備份為 promptbox.db.plain.bak，資料不自動遷移，可手動匯入。

v2.2.0 – v2.3.0

2026-05-26 ~ 2026-05-28

MCP 整合登場——你的 Prompt 資產第一次能被 IDE 直接取用。

  • 🔌內建 MCP 伺服器：Cursor、VS Code、Claude Desktop 可直接讀取 Prompt（SSE + Streamable HTTP 雙傳輸）
  • 🛡️Token 認證：自動生成隨機 Token 保護本機服務
  • 🖥️System Tray 模式：關閉視窗後縮小至系統匣，MCP 服務持續運行
  • 🎛️MCP 設定面板：查看狀態、管理 Token、一鍵產生 IDE 設定 JSON

v2.0 – v2.1.3

2026-03-09 ~ 2026-03-24

體驗打磨期：更快的啟動、更自由的安裝、更聰明的資料維護。

  • 🔄自動更新機制：啟動時自動比對 GitHub Releases 最新版本（v2.0）
  • ⚡極速啟動：Skeleton 載入畫面與非同步資料載入，開啟即用（v2.1.0）
  • 📂自訂安裝路徑 與 免安裝便攜版（.zip）（v2.1.2）
  • 🧹清除快取與重設：設定頁一鍵還原出廠狀態（v2.1.2）
  • 🔑孤立變數自動清理：儲存時自動移除內容中已不存在的變數（v2.1.3）

v1.0 – v1.6

2026-02-15 ~ 2026-02-16

一切的起點：把 Prompt 收好、找得到、一鍵複製。

  • 📁基礎 Prompt 管理：資料夾分類、標籤系統、本地儲存（v1.0）
  • 📝Markdown 預覽：長 Prompt 也好讀（v1.2）
  • 🎨變數樣板：{{variableName}} 動態替換，一份樣板重複用（v1.5）
  • 📖內建教學：教學資料夾與防呆機制，無痛上手（v1.6）
