Get Started

3 步驟，1 分鐘開始

1

下載並安裝

從 GitHub Releases 下載對應平台的安裝檔，依精靈指示完成安裝（Windows 可自選安裝路徑）。

2

首次啟動

App 會自動建立加密資料庫並載入「教學資料夾」，跟著範例卡片認識介面，不怕弄壞資料。

3

複製第一條 Prompt

新增一張卡片、點一下複製按鈕，貼進 ChatGPT / Claude——你已經完成第一次效率升級。

Notes

安裝小提示

update

更新方式：v3.0 起採手動更新。App 啟動時偵測到新版本會提示你前往下載頁，不會無聲自動安裝，你的資料由你掌控。

lock

資料加密：首次啟動會建立 SQLCipher 加密資料庫，金鑰由作業系統的 Keychain（Windows DPAPI / macOS Keychain）保護，不落明文。

warning

從 v2.x 升級：首次啟動 v3 會將舊的明文資料庫備份為 promptbox.db.plain.bak 並建立全新加密資料庫，舊資料不會自動遷移，可從備份手動匯入。
