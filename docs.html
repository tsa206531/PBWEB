操作手冊

從安裝到進階的機密保險庫與 MCP 整合，這份手冊帶你把 PromptBox 用到淋漓盡致。

rocket_launch快速開始

安裝

  1. 前往 下載頁 取得對應平台的安裝檔（Windows 提供安裝版與免安裝便攜版）。
  2. 執行安裝精靈。Windows 版可自訂安裝磁碟與資料夾。
  3. 啟動 PromptBox。

首次啟動

第一次開啟時，App 會自動完成兩件事：

  • 建立加密資料庫：所有資料以 SQLCipher 靜態加密儲存在你的電腦，金鑰由作業系統的 Keychain 保護。
  • 載入教學資料夾：側欄會出現內建的教學卡片，跟著範例操作即可熟悉介面——教學內容與你的真實資料隔離，怎麼玩都不怕弄壞。
lightbulb

建議的第一步：新增一張你最常用的 Prompt 卡片，點複製按鈕貼到 ChatGPT / Claude 用一次。感受到「1 秒取用」之後，再慢慢把其他 Prompt 搬進來。

style基本操作

新增卡片

  1. 點擊主畫面的「新增 Prompt」按鈕。
  2. 填寫 標題、內容（支援 Markdown），並可加上 說明、標籤、選擇 資料夾。
  3. 儲存後卡片會出現在卡片牆上，左下角顯示 #id 編號（方便在 MCP 或回報問題時引用）。

一鍵複製

每張卡片右下角都有複製按鈕，點一下即把完整內容送進剪貼簿，圖示會短暫變成 ✓ 表示成功。這就是 PromptBox 最核心的動作：找到卡片 → 點一下 → 貼上就用。

搜尋

頂部搜尋框可同時比對 標題、內容、標籤。Prompt 破百條之後，搜尋是最快的入口。

我的最愛

點卡片上的 ⭐ 星號把常用 Prompt 加入「我的最愛」，側欄可一鍵過濾。

檢視模式

右上角可切換 網格檢視（預設）與 列表檢視；Markdown 內容可切換編輯 / 預覽分頁。

folder資料夾與標籤

資料夾

  • 在側欄「資料夾」區點「+」新增資料夾，把不同用途的 Prompt 分開收納（例如：Code Review、行銷文案、會議記錄）。
  • 資料夾可重新命名、刪除；刪除前會跳出確認對話框防止誤刪。
  • 免費版最多可建立 5 個資料夾，個人版無上限。

標籤

  • 每張卡片可掛多個標籤，側欄的標籤區支援 多選過濾（同時選多個標籤縮小範圍）。
  • 選了一堆標籤想重來？點標籤區的「重設」按鈕一鍵清空。
lightbulb

分類心法：資料夾放「用途」（專案、場景），標籤放「屬性」（語言、模型、口吻）。一張卡片只能在一個資料夾，但可以有多個標籤。

data_object變數樣板

變數樣板讓一份 Prompt 適用無限情境。在內容中以雙大括號宣告變數：

請扮演資深 {{角色}}，用 {{語氣}} 的口吻，
針對以下主題寫一篇 {{字數}} 字的文章大綱：

主題：{{主題}}

使用方式

  1. 儲存卡片後，右側會出現 變數面板，自動列出內容中的所有 {{變數}}。
  2. 在面板填入當次的值（可為每個變數設定預設值）。
  3. 點複製時，變數會自動代入你填的值；沒填的變數保留 {{佔位符}} 原樣。

強型別設定（進階）

變數預設為字串型別。開啟強型別模式後，可為每個變數指定型別（數字、布林、列舉等），在 MCP 整合時讓 AI 帶入的參數受到 JSON Schema 驗證。

編輯內容時若刪掉了某個 {{變數}}，儲存時 App 會自動清掉對應的孤立變數值，不需手動整理。

lock機密卡片與資料鎖

PromptBox 提供兩層保護，讓含有機密內容（API Key、內部規範、個資範本）的卡片安全存放：

機制作用適合放什麼
資料鎖 🔒一鍵鎖定，卡片對 AI（MCP）完全隱藏，GUI 仍可直接查看不想被 AI 讀到、但自己常看的內容
機密卡片 🔐以自訂密碼做卡片級 AES-256 加密，GUI 也要輸入密碼才能查看API Key、密碼、真正的機密

建立機密卡片

  1. 編輯卡片時開啟「機密」選項，設定一組自訂密碼。
  2. 之後查看這張卡片都需要輸入密碼解鎖；所有密碼欄位都有 👁️ 顯示/隱藏切換，輸入前可先確認沒打錯。
  3. 解鎖後變數面板照常可用：解析 {{變數}}、填值、複製代入一氣呵成。
report

忘記密碼將永久無法復原內容。密碼不會被儲存在任何地方，也沒有後門或重設機制。設定前請務必確認自己記得住這組密碼。

機密釋放（Consent）

預設情況下 AI 永遠讀不到機密卡片。若你希望特定機密卡片可以「在你授權下」餵給 AI，可設定釋放策略：

  • never（預設）：永不釋放，AI 完全看不到這張卡的存在。
  • once：AI 每次請求都需要你輸入密碼授權，用完即棄。
  • session：授權一次，本次 App 執行期間有效。

每次釋放（允許 / 拒絕 / 逾時）都會寫入釋放稽核紀錄，可在 MCP 面板檢視，全程可追溯。

link卡片串聯

把多張卡片組合成一份大 Prompt。在側欄的內建「串聯資料夾」中建立組合卡，用以下語法引用其他卡片：

{{@prompt:12}}   ← 引用 #12 卡片的內容

再加上這段自己的補充說明，
{{@prompt:34}}   ← 再引用 #34

運作規則

  • 引用支援遞迴展開（被引用的卡片裡還可以再引用別張），深度上限 5 層，並自動偵測循環引用。
  • 輸入時即時展開預覽，Preview 分頁顯示「展開 + 填值後」的最終結果。
  • 被引用卡片的變數會取聯集出現在組合卡的變數面板；已填的值自動帶入當預設，同名不同值的衝突會留空並標示提醒。
  • 鎖定卡與機密卡無法被串聯引用——機密不會經由串聯間接洩漏。
  • 組合卡永遠只儲存 {{@prompt:N}} 原始 token，展開結果不寫回內容。

extensionMCP 整合

PromptBox 內建 MCP（Model Context Protocol）伺服器，讓 Cursor、VS Code、Claude Desktop 等工具直接取用你的 Prompt 資產——每張未鎖定的卡片都自動成為一個可呼叫的 Tool。

啟動伺服器

  1. 點擊側欄的 MCP 設定 按鈕開啟面板。
  2. 啟動伺服器（僅監聽本機 127.0.0.1，不對外開放）。
  3. 建立一組 具名 Token（可建多組、可個別撤銷，每組限速 60 req/min）。
  4. 點「產生 IDE 設定」複製現成的設定 JSON。

IDE 設定範例

{
  "mcpServers": {
    "promptbox": {
      "url": "http://127.0.0.1:<port>/mcp",
      "headers": {
        "Authorization": "Bearer <你的Token>"
      }
    }
  }
}

支援 Streamable HTTP（/mcp）與 legacy SSE 兩種傳輸方式。連線後，AI 可以：

  • 列出並呼叫你的 Prompt Tool（prompt_{id}），帶入變數參數，取得編譯後的純淨字串——比整段貼上省 Token。
  • 缺少必填變數時收到明確的引導訊息，會回頭向你要參數。
  • 你新增 / 編輯 / 鎖定卡片時，IDE 端的 Tool 列表即時刷新。
visibility_off

隱私邊界：鎖定卡片（🔒）與未授權的機密卡片（🔐）對 MCP 完全隱藏——AI 不僅讀不到內容，連卡片的存在都看不到。

System Tray 常駐

關閉主視窗後 App 會縮小到系統匣繼續運行，MCP 服務不中斷；要完全退出請在系統匣圖示上按右鍵選擇結束。

settings設定與疑難排解

主題與語言

  • 深色 / 淺色模式：點頂部的主題切換按鈕，或跟隨系統設定自動切換。
  • 介面語言：支援繁體中文、English、日本語，可在設定頁切換。

更新版本

App 啟動時會自動比對 GitHub Releases 的最新版本，發現新版會顯示更新提示，點擊後前往下載頁手動下載安裝——不會無聲自動更新，你的資料由你掌控。

清除快取與重設

設定頁提供「清除快取與重設」功能，可一鍵清除所有資料並還原出廠狀態。

warning

重設無法復原。執行前請先確認重要的 Prompt 已另行備份（個人版可用匯出功能）。

常見問題

  • 從 v2.x 升級後看不到舊資料？v3 首次啟動會把舊明文資料庫備份為 promptbox.db.plain.bak 並建立新的加密資料庫，舊資料不自動遷移，可從備份手動匯入。
  • 機密卡片密碼打不開？先用密碼欄位的 👁️ 顯示切換確認輸入內容與大小寫。密碼錯誤沒有重試上限，但也沒有任何找回機制。
  • IDE 連不上 MCP？確認伺服器已啟動、Port 與設定 JSON 一致、Token 未被撤銷；收到 429 表示該 Token 超過每分鐘 60 次的速率限制。
